SilverStripe, MailChimp, Web Design

Custom D had been working on an in-house task management app for quite a while and it was now time to gather public interest about the upcoming Beta launch of the application.

We designed a one-page SilverStripe website to attract beta testers which gave details about the product on offer and the range of benefits that it would provide small businesses.

In order to gather public interest, a simple input field was added to the site for users to subscribe. The input field linked to a MailChimp mail list where users were regularly updated with the latest information regarding the Beta release of the app.
